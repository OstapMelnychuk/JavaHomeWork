Папка для попередніх домашніх завдань.
Для того щоб закинути сюди домашнє завдання, створіть папку названу за вашим ім'ям та прізвищем.
В ній створіть текстовий документ в якому вкажіть:
1) Тему завдання(За яку тему ви виконуєте завдання, приклад String methods).
2) Тему вашої домашньої роботи (На яку тему ви виконуєте завдання, приклад Шифрування Цезаря)

Якщо домашніх завдань декілька то в папці з ім'ям та прізвищем зробіть для них окремі папки.
